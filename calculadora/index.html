O que é IMC?

O Índice de Massa Corporal (IMC) é um cálculo utilizado para avaliar se uma pessoa está dentro do peso ideal, levando em consideração seu peso e altura.

Ele é amplamente utilizado por profissionais de saúde como uma ferramenta simples para identificar possíveis problemas relacionados ao peso.
